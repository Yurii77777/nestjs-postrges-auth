# nestjs-postrges-auth

## How to start

1. Istall PostreSQL;
2. Create `.env` file contains:

- `DATABASE_HOST`;
- `DATABASE_PORT`;
- `DATABASE_USERNAME`;
- `DATABASE_PASSWORD`;
- `DATABASE_NAME`;

3. yarn install or npm i
4. npm run start

### Additionally

Є можливість винести перевірку e-mail на унікальність написавши кастомний декоратор та використовувати його в dto для валідації.
